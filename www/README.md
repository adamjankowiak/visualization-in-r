# www/

Katalog zawiera pliki statyczne serwowane przez Shiny:

- interaktywną mapę Plotly wygenerowaną w `project1_country_names_pl.R`,
- ranking Plotly,
- statyczne mapy PNG dla EUR i PPP.

Plik `ikea_price_boxplot_plotly_eur.html` nie jest dołączony do paczki, ponieważ nie był dostępny w przesłanych plikach. Skrypt `R/prepare_dashboard_data.R` przeniesie go tutaj automatycznie, jeżeli zostanie wygenerowany lokalnie z pełnego datasetu.
